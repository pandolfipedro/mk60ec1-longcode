MK60EC1 Long Code Editor

Jetta 2009–2016 · edição byte a byte

Como usar
  1. Cole o VIN e escolha o part number do módulo (CC/BL/AD)
  2. Clique Calcular (ou Importar hex se já tem coding)
  3. Ajuste bytes 16+ se necessário
  4. Compare com o hex lido no VCDS (campo opcional abaixo)
  5. Copie e grave no 03-ABS · login 20795
Termos comuns
G85
Sensor de ângulo do volante - o ESP usa para saber para onde você está virando.
Espelho
Byte calculado automaticamente a partir de outro (ex.: byte 14 espelha o 6).
VIN (bytes 1–13)
Parte do long code derivada do chassi - recalculada ao informar o VIN.
HHC
Hill Hold Control - segura o carro em ladeira por alguns segundos.
PLA (byte 17 ABS)
Assistente que estaciona sozinho (módulo J791, PLA 2.0+) - NÃO é sensor que apita. Jetta BR com 8K/OPS tem sensores no módulo 09, PLA aqui fica OFF.
ACC
Cruise adaptativo - radar dianteiro; desmarque se o carro não tiver.
XDS
Controle de tração extra em curva - comum no CC.
MCB
Multi-Collision Braking - frenagem automática pós-colisão via módulo airbag.
AK / byte 18
Sensores ABS direcionais (ativos) - só com PLA 2.0 ou ACC. Errado = DTC de roda.
EPS / J500
Direção elétrica - G85 integrado na caixa de direção (não na coluna).
Long code
Sequência hex gravada no módulo 03 via Coding-07 no VCDS.
MK60EC1
Família ABS/ESP 1K0907379 usada no Jetta importado BR.

Entrada

Long code

Bytes diferentes ficam destacados no editor.

Confira bytes 16+ antes de gravar no VCDS.
Gravar no VCDS
  1. Conectar VCDS → chassi → 03 - ABS/ESP
  2. Coding-07 · login 20795 (CC/BM+)
  3. Colar hex sem espaços
  4. Conferir byte a byte · Basic Settings G85 após troca
Migrar long code

Rotas comuns BRmais comum: AD→CC, AD→BL, BJ→BL, BJ→CC, BL→CC. AH ≈ AD.

Após migrar, o resultado é aplicado automaticamente no editor abaixo.
