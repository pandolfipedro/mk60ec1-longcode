How to write in VCDS
  1. Connect VCDS → select chassis (VIN) → 03 — ABS/ESP
  2. Coding-07 (long coding)
  3. If login is required: 20795 (CC/BM+ modules)
  4. Paste the long code — continuous hex, no spaces, e.g. 113B400D4A25…8060
  5. Use o Long Coding Helper to review byte by byte — Byte 0 = left
  6. Check in the Helper before Do It!
  7. After module swap: Basic Settings G85 (steering angle sensor)

In VCDS each byte = 2 hex digits (Byte 0 = first 2 characters). Copy and paste the full string — Ross-Tech recommends copy/paste for long coding.
